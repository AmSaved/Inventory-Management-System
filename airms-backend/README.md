# Asset & Inventory Request Management System (AIRMS)

## 🚀 Overview
AIRMS is a comprehensive enterprise-grade asset management system designed for multi-branch organizations to manage internal company assets with strict approval workflows.

## 📋 Features
- Multi-branch inventory tracking
- Strict approval hierarchy (Chairman → Storage Manager)
- Permission-based access control
- Complete asset lifecycle management
- Barcode/QR code generation
- Real-time notifications
- Audit logging
- Report generation

## 🛠 Technology Stack
- **Backend**: Node.js + Express.js
- **Database**: PostgreSQL with Sequelize ORM
- **Authentication**: JWT with refresh tokens
- **Real-time**: Socket.io
- **Queue**: Bull (Redis)
- **Testing**: Jest
- **Documentation**: Swagger/OpenAPI
